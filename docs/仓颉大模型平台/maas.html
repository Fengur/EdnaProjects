基础接口

聊天对话接口

POST /v1/chat/completions

用于与模型进行多轮对话，支持流式输出。

请求参数

{
  "model": "cangjie-8b-v2.1.0",
  "messages": [
    {
      "role": "user",
      "content": "你好，请介绍一下汽车保养的基本知识"
    }
  ],
  "temperature": 0.7,
  "max_tokens": 2048,
  "stream": false
}

响应示例

{
  "id": "chatcmpl-123456789",
  "object": "chat.completion",
  "created": 1677652288,
  "model": "cangjie-8b-v2.1.0",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "汽车保养是确保车辆正常运行和延长使用寿命的重要环节..."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 15,
    "completion_tokens": 120,
    "total_tokens": 135
  }
}

文本生成接口

POST /v1/completions

用于文本补全和生成任务。

请求参数

{
  "model": "cangjie-8b-v2.1.0",
  "prompt": "请写一段关于新能源汽车的介绍：",
  "max_tokens": 500,
  "temperature": 0.8
}

SDK示例

Python SDK

import requests

url = "https://api.cangjie.ai/v1/chat/completions"
headers = {
    "Authorization": "Bearer YOUR_APP_KEY",
    "Content-Type": "application/json"
}

data = {
    "model": "cangjie-8b-v2.1.0",
    "messages": [
        {"role": "user", "content": "你好"}
    ]
}

response = requests.post(url, headers=headers, json=data)
print(response.json())

JavaScript SDK

const response = await fetch('https://api.cangjie.ai/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_APP_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'cangjie-8b-v2.1.0',
    messages: [
      { role: 'user', content: '你好' }
    ]
  })
});

const data = await response.json();
console.log(data);
